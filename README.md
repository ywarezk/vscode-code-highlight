# Academeez: VSCode extension - Lecture Assistant

Enhance your live coding lectures with powerful code highlighting and annotation tools. **Academeez: Code Lecture Assistant** helps programming lecturers create more engaging and educational coding sessions by allowing you to highlight specific code sections and add contextual comments in a dedicated side panel.

Perfect for instructors who want to draw attention to key concepts, explain complex logic, or provide additional context during live demonstrations. Keep your audience focused and make your code explanations crystal clear.
